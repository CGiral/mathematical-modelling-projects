# Algebraic and Differential Geometry

Selected academic work in algebraic and differential geometry.

## Contents

This section includes two mathematical reports and a SageMath notebook used to support symbolic computations in the algebraic geometry work.

The algebraic geometry report focuses on polynomial ideals, Gröbner bases, elimination theory and algebraic varieties. The differential geometry report focuses on differentiable manifolds, diffeomorphisms, vector fields, integral curves and differential forms.

## Topics

* Polynomial ideals
* Gröbner bases
* Buchberger's algorithm
* Ideal membership
* Elimination theory
* Algebraic varieties
* Differentiable manifolds
* Diffeomorphisms
* Vector fields
* Integral curves
* Differential forms
* Symbolic computation with SageMath

## Project Files

* `algebraic_geometry_solved_problems.pdf`: solved problems in algebraic geometry and Gröbner bases.
* `algebraic_geometry_sagemath.ipynb`: SageMath notebook used for symbolic computations. *(To be added)*
* `differential_geometry_solved_problems.pdf`: solved problems in differential geometry.

## Academic Context

These works were developed as coursework for the MSc in Mathematical Modelling, Statistics and Computing.

The published version preserves the original mathematical developments and computations. Changes are limited mainly to repository organisation, file naming and presentation.
