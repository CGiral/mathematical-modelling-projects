# Algebraic and Differential Geometry

Selected academic work in algebraic and differential geometry.

## Contents

This section includes two mathematical reports and two SageMath notebooks used to support symbolic computations in the algebraic geometry work.

The algebraic geometry report focuses on polynomial ideals, Gröbner bases, elimination theory and algebraic varieties, combining formal mathematical developments with SageMath computations.

The differential geometry report focuses on differentiable manifolds, diffeomorphisms, vector fields, integral curves and differential forms.

## Topics

* Polynomial ideals
* Gröbner bases
* Buchberger's algorithm
* Ideal membership
* Elimination theory
* Algebraic varieties
* Symbolic computation with SageMath
* Differentiable manifolds
* Diffeomorphisms
* Vector fields
* Integral curves
* Differential forms

## Project Files

* `algebraic_geometry_solved_problems.pdf`: solved problems in algebraic geometry and Gröbner bases.
* `algebraic_geometry_sagemath_distance_formulation.ipynb`: SageMath notebook for the distance-based algebraic formulation.
* `algebraic_geometry_sagemath_oriented_formulation.ipynb`: SageMath notebook for the oriented algebraic formulation over Q(sqrt(3)).
* `differential_geometry_solved_problems.pdf`: solved problems in differential geometry.

## Academic Context

These works were developed as coursework for the MSc in Mathematical Modelling, Statistics and Computing.

The published version preserves the original mathematical developments, symbolic computations and results. Changes are limited mainly to repository organisation, file naming and presentation.
