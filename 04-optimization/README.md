# Optimization

Selected academic work in mathematical optimization, integer programming and stochastic programming.

## Contents

This section includes theoretical and computational projects involving integer programming, combinatorial optimization, algebraic methods and stochastic optimization.

The projects combine mathematical modelling, algorithmic reasoning, computational experiments and optimization notebooks.

## Topics

* Integer programming
* Combinatorial optimization
* Nonlinear optimization
* Stochastic optimization
* Karush–Kuhn–Tucker conditions
* Polyomino packing
* Risk-neutral optimization
* Worst-case optimization
* Value-at-Risk models
* Gröbner bases and integer programming
* Toric ideals
* Julia
* JuMP
* Python
* Mathematical programming

## Project Files

* `polyomino_packing.ipynb`: Julia/JuMP notebook modelling a polyomino packing problem as a binary integer programming problem.
* `groebner_bases_integer_programming.pdf`: report connecting Gröbner bases, toric ideals and integer programming.
* `stochastic_optimization.ipynb`: computational notebook involving stochastic optimization models.
* `stochastic_optimization.pdf`: written report associated with the stochastic optimization work.

## Academic Context

These works were developed as coursework for the MSc in Mathematical Modelling, Statistics and Computing.

The published version preserves the original mathematical developments, code, models and results. Changes are limited mainly to repository organisation, file naming and presentation.
