# Nonlinear Dynamics

Selected academic work in nonlinear dynamical systems.

## Contents

This section includes a mathematical report focused on the qualitative and analytical study of nonlinear dynamical systems.

The work covers equilibrium analysis, stability, linearisation, bifurcations, periodic orbits and qualitative behaviour of nonlinear systems.

## Topics

* Nonlinear dynamical systems
* Equilibrium points
* Linearisation
* Stability analysis
* Phase portraits
* Bifurcations
* Periodic orbits
* Poincaré–Bendixson theory
* Lyapunov-type arguments
* Nonlinear oscillations
* Chaotic dynamical systems

## Project Files

* `nonlinear_dynamics_and_applications.pdf`: solved problems and analytical developments in nonlinear dynamics.

## Academic Context

This work was developed as coursework for the MSc in Mathematical Modelling, Statistics and Computing.

The published version preserves the original mathematical developments, computations and results. Changes are limited mainly to repository organisation, file naming and presentation.
