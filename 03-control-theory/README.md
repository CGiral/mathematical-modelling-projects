# Control Theory

Selected academic work in control theory and dynamical systems modelling.

## Contents

This section includes a MATLAB-based project focused on modelling drug administration as a dynamical system.

The work combines physical modelling, differential equations, state-space representation, transfer functions and numerical simulation.

## Topics

* Control theory
* Dynamical systems
* Mass-balance modelling
* State-space representation
* Transfer functions
* Impulse response
* Pulse input modelling
* Steady-state analysis
* Stability
* MATLAB simulation

## Project Files

* `drug_administration_control.pdf`: complete report describing the mathematical model, control-theoretic formulation and simulation results.
* `drug_administration_control.mlx`: original MATLAB Live Script used for the computations and simulations.

## Academic Context

This work was developed as coursework for the MSc in Mathematical Modelling, Statistics and Computing.

The published version preserves the original mathematical developments, code, simulations and results. Changes are limited mainly to repository organisation, file naming and presentation.
