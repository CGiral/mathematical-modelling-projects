# Mathematical Modelling Projects

A collection of selected projects developed during my MSc in Mathematical Modelling, Statistics and Computing.

The repository brings together theoretical and computational work in geometry, nonlinear dynamics, control theory and mathematical optimization. The projects combine formal mathematical reasoning with symbolic computation, numerical methods, simulation and mathematical programming.

## Areas

### Algebraic and Differential Geometry

Work involving algebraic varieties, polynomial ideals, Gröbner bases, elimination theory and differentiable manifolds.

The algebraic geometry project combines detailed mathematical developments with symbolic computations performed in SageMath. The original SageMath notebook used to support and verify the calculations is included alongside the complete written report.

Main topics:

* Polynomial ideals
* Gröbner bases
* Buchberger's algorithm
* Ideal membership
* Elimination theory
* Algebraic varieties
* Differentiable manifolds
* Diffeomorphisms
* Vector fields
* Integral curves
* Differential forms

### Nonlinear Dynamics

Analysis of nonlinear differential systems through qualitative and analytical methods.

Main topics:

* Equilibrium points
* Linearisation
* Stability analysis
* Bifurcations
* Periodic orbits
* Poincaré–Bendixson theory
* Lyapunov-type arguments
* Nonlinear oscillations
* Chaotic dynamical systems

### Control Theory

Mathematical modelling and simulation of dynamical systems using state-space methods and transfer functions.

The control project studies the administration of medication through a two-compartment model, combining physical modelling, differential equations and MATLAB simulation.

Main topics:

* Mass-balance modelling
* State-space representation
* Transfer functions
* Impulse response
* Realistic input modelling
* Steady-state analysis
* Stability
* Time-domain simulation

### Mathematical Optimization

The repository includes theoretical and computational work in continuous, integer, nonlinear and stochastic optimization.

Main topics:

* Linear programming
* Integer programming
* Branch-and-Bound
* Branch-and-Cut
* Branch-and-Price
* Branch-and-Cut-and-Price
* Karush–Kuhn–Tucker conditions
* Nonlinear optimization
* Stochastic optimization
* Risk-neutral, worst-case and Value-at-Risk models
* Travelling Salesman Problem
* Polyomino packing
* Algebraic approaches to integer programming
* Gröbner bases and toric ideals
* Mathematical programming models in MPS format
